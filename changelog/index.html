重要通知：
自 v3.0.0 起，原 Hexo 主题 ILS 改名为 Keep， v3.0.0 在原 ILS 基础上做了大量改进、优化和代码重构，同时也新增许多功能，强烈建议使用最新版本 Keep 主题及最新版本的主题配置文件 _config.yml。

v3.0.3 [2020-12-01]

  1. 修复用户无法使用自己图片作为 头像(avatar) 和 网站图标(favicon) 的 Bug。

  2. 主题配置文件增加 favicon 和 avatar 配置选项，可使用本地图片或 CDN 外链。

  3. 重构 CSS 字体单位，em -> rem，更合理，更协调，更优雅。

v3.0.0 [2020-11-26]

  1. 增加首屏展示功能。

  2. 增加第三方社交链接功能。

  3. 增加在线字号调整功能。

  4. 重构右下角侧边工具栏。

  5. 增加自由切换内容区域宽度功能。

  6. 升级 fontawesome 图标库。

  7. 增加设置 CDN 加速功能。

  8. 增加底部起始年份设置功能。

  9. 优化 Markdown 样式。

  10. 移除原 ILS 的 magic 配置项 和 normal 风格。

  11. 美化友链页面并采用 _data 来存储。

  12. 首页文章块显示该文章创建于多久之前。

  13. 文章内容页增加版权信息模块。

  14. ……

v2.1.3 [2020-10-30]

  1. 文章顶置的属性改为 sticky，无需安装第三方插件。

  2. 新创建的页面添加评论功能的属性改为 comment。
    _config.yml 主题配置文件的评论属性也需要修改 comments -> comment，强烈建议使用最新版的 _config.yml。

  3. 文章内容页增加博主头像图片和Lv标识，Lv等级根据文章数量判断。

  4. 优化 Gitalk 评论插件，适配夜间模式。
